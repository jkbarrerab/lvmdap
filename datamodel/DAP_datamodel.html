Datamodel: dap.fits.gz

General Description

The dap.fits.gz are the FITS files that comprises the dataproducts derived using LVM Data Analysis Pipeline for each LVM CFrame generated by the LVM-DRP. Each FITS file comprises eight extensions. The first extension is just the header of the original LVM CFrame, while the remaining extensions correspond to a set of FITs tables including different sets of dataproducts described below, with each row corresponding to a certain science fiber of the LVM IFUs. Each fiber is identified by the same primary key that it is included (and shared) in each fiber, labeled as id, formed by the LVM unique index of each exposure and the original index of the fiber in the SLITMAP, adopting the following scheme: id=exposure.fiberid.

The files are stored temporarely at http://ifs.astroscu.unam.mx/LVM/LVMDAP/v0.231130/

We recommend to read this file using the read_DAP_file included in the lvmdap package, available at the lvmdap repository

We include, in addition to the name of the column in the DAP FITS file for each extension (Key) the assigned name of the colum in the DAP table once readed using this code (tab_DAP_Key)

Naming Convention
dap-[DAP_CONF]-[LVM_FILE_ID].dap.fits.gz, where [LVM_FILE_ID] is the LVM unique ID for a certain exposure, [DAP_CONF] is a metacode identifying the DAP run.

Sections
  • HDU0: PRIMARY Header of the original LVM CFrame fits file.
  • HDU1: PT Position Table of the science fibers in the LVM CFrame fits file.
  • HDU2: RSP Main parameters derived from the analysis of the stellar populations, including the LW average of the stellar properties (Teff, log_g, [Fe/H], [alpha/Fe]), the dust attenuation (Av), and the kinematics properties.
  • HDU3: COEFFS Weights of the descomposition of the stellar population for the adopted RSP library.
  • HDU4: PM_ELINES Flux intensities for the strongest emission lines in the considered wavelengh range, together with the kinematics properties of Ha, derived based on a Gaussian fitting of each emission line.
  • HDU5: NP_ELINES_B Main parameters of a set of strong and weak emission lines covered by the Blue arm of the spectrograph derived using a weighted momentum analysis based on the kinematics of Halpha. It includes the flux intensity, equivalent width, velocity and velocity dispersion, and the corresponding errors for the different analyzed emission lines.
  • HDU6: NP_ELINES_R Main parameters of a set of strong and weak emission lines covered by the Red arm of the spectrograph derived using a weighted momentum analysis based on the kinematics of Halpha. It includes the flux intensity, equivalent width, velocity and velocity dispersion, and the corresponding errors for the different analyzed emission lines.
  • HDU7: NP_ELINES_I Main parameters of a set of strong and weak emission lines covered by the Infrared arm of the spectrograph derived using a weighted momentum analysis based on the kinematics of Halpha. It includes the flux intensity, equivalent width, velocity and velocity dispersion, and the corresponding errors for the different analyzed emission lines.
  • HDU8: INFO Set of parameters adopted by the DAP for the current analysis, including the adopted RSP libraries.
  • TAB_DAP Description of the astropy Table generated by the read_DAP_file code
